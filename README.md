AI FAQ Chatbot

AI FAQ Chatbot is a full-stack AI chatbot application built using React.js, FastAPI, Groq API, Google Authentication, and PostgreSQL cloud database.

The chatbot can be used for college FAQs, HR support, customer support, and product assistance. It provides real-time AI responses, user authentication, Google login, chat history, and a ChatGPT-style user interface.

Features

- AI-powered chatbot
- Email signup and login
- Continue with Google authentication
- JWT-based authentication
- Protected chatbot page
- ChatGPT-style interface
- Dark mode and light mode
- Chat history sidebar
- Delete chat history
- User-wise chat storage
- Cloud PostgreSQL database using Neon
- FastAPI backend
- React.js frontend

Technologies Used

Frontend:
- React.js
- Vite
- Tailwind CSS
- Axios
- React Router
- Google OAuth

Backend:
- FastAPI
- Python
- SQLAlchemy
- PostgreSQL
- JWT Authentication
- Groq API
- Google Auth Verification

Database:
- Neon PostgreSQL
